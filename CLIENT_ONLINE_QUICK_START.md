# Инструкция для клиента: подключение к онлайн-базе

Эта инструкция нужна для компьютера клиента.

## 1. Установить Tailscale

Скачайте и установите Tailscale:

- [https://tailscale.com/download](https://tailscale.com/download)

После установки войдите в тот аккаунт или tailnet, который вам выдаст владелец базы.

## 2. Убедиться, что Tailscale подключен

В приложении Tailscale должен быть статус:

```text
Connected
```

## 3. Запустить программу

Откройте приложение и на стартовом экране выберите:

```text
Онлайн база
```

## 4. Ввести параметры подключения

Если поля не заполнились автоматически, введите:

```ini
host = 100.86.4.84
port = 5432
name = cost_online_demo
user = cost_client_app
password = CostClientApp_2026!
sslmode = disable
```

## 5. Подключиться

Нажмите кнопку подключения в приложении.

Если все настроено верно, программа откроется в обычном режиме и будет работать уже с онлайн-базой.

## Если подключение не удалось

Проверьте:

1. Tailscale на клиентском компьютере в статусе `Connected`.
2. Серверный компьютер включен.
3. На серверном компьютере запущен PostgreSQL.
4. Введен адрес `100.86.4.84`, а не `localhost`.
5. Логин `cost_client_app` и пароль введены без ошибок.
