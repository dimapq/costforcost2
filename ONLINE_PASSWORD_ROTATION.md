# Безопасная смена пароля для клиента

Теперь пароль онлайн-подключения можно менять без правки кода и без ручного редактирования файлов.

## Где это делается

В приложении:

1. Откройте `Настройки`.
2. Нажмите `Показать параметры онлайн-подключения`.
3. Нажмите `Сменить пароль`.

## Что произойдет автоматически

После подтверждения приложение:

1. изменит пароль пользователя онлайн-базы;
2. обновит онлайн-профиль в основном `config.ini`;
3. подготовит новый клиентский файл `client_config_online.ini`.

## Правильный порядок действий

1. Смените пароль через интерфейс.
2. Возьмите новый файл из папки `exports`.
3. Передайте клиенту только новый `client_config_online.ini`.
4. Попросите клиента заменить старый файл настроек новым.

## Что не нужно делать

- Не редактировать пароль вручную в коде.
- Не давать клиенту пользователя `postgres`.
- Не отправлять старый `config.ini`, если пароль уже поменялся.

## Что важно помнить

Старый клиентский конфиг после смены пароля перестанет подходить.
Поэтому после каждой смены пароля клиенту нужно выдавать новый файл настроек.
