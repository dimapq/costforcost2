# MachineCost Pro v1.3

Desktop-приложение для учета производства станков, материалов, сотрудников, финансов и себестоимости.

Проект помогает вести производственный учет в одном месте:
- считать себестоимость станков;
- учитывать материалы и склад;
- фиксировать рабочее время сотрудников;
- учитывать продажи и прибыль;
- распределять косвенные расходы;
- контролировать налоги и денежные операции.

## Возможности

### Станки
- учет станков в производстве;
- учет готовых станков на складе;
- учет проданных станков;
- дата начала и окончания производства;
- расчет количества дней от запуска до продажи;
- просмотр деталей себестоимости;
- редактирование готовых и проданных станков;
- отмена производства.

### Склад и материалы
- добавление материалов вручную;
- полное редактирование материала, а не только цены;
- учет источника, примечания и даты обновления;
- поддержка FIFO;
- учет материалов с пометкой `наличка`;
- корректировка остатков;
- перевод плит и листовых материалов в готовые детали;
- отдельная вкладка для раскроя плит;
- история, из какой плиты были изготовлены детали;
- повторное изготовление уже существующей детали из выбранной плиты.

### Сотрудники
- список сотрудников и их ставок;
- расчет зарплаты по периоду;
- расчет премий по проценту;
- учет даты последней выплаченной премии;
- учет перерасчетов по зарплате;
- учет услуг сотрудников отдельно от основной зарплаты;
- фиксация переплат и недоплат.

### Финансы
- отчет о прибыли и убытках;
- расчет налогов по всем безналичным операциям;
- дата последней уплаты налога;
- учет косвенных ежемесячных расходов:
  - аренда,
  - электричество,
  - интернет,
  - домен,
  - и другие постоянные расходы;
- автоматическое распределение косвенных расходов по станкам по дням производства;
- учет дней простоя и накопленных косвенных расходов в простой.

### Операции
- быстрый ввод рабочих часов;
- последние действия в программе;
- журнал операций по основным действиям пользователя.

### Экспорт и обслуживание
- экспорт всей базы в Excel;
- SQL-дамп базы данных;
- настройки подключения к базе через интерфейс;
- сборка готового Windows-приложения.

## Как работает расчет косвенных расходов

Косвенные расходы распределяются по дням.

Логика:
1. Для каждой категории задается месячная сумма.
2. Месячная сумма делится на количество дней в месяце.
3. Для каждого дня определяется, сколько станков было в производстве.
4. Дневная сумма делится поровну между всеми станками, которые производились в этот день.
5. Полученная сумма добавляется в себестоимость каждого станка.

Если в какой-то день производство не велось, этот день учитывается как простой.

## Запуск из исходников

### 1. Установить зависимости
```bash
python -m venv .venv
.venv\Scripts\python.exe -m pip install -r requirements.txt
```

### 2. Настроить подключение к базе
Заполнить `config.ini` или указать параметры подключения при первом запуске приложения.

### 3. Запустить приложение
```bash
start.bat
```

или

```bash
.venv\Scripts\python.exe -m frontend.main
```

## Сборка Windows-версии

Сборка выполняется через `PyInstaller`.

```bash
.venv\Scripts\python.exe -m PyInstaller MachineCostPro.spec
```

После сборки готовая версия появляется в папке:

```text
dist/MachineCostPro
```

Готовый архив для отправки:

```text
dist/MachineCostPro.zip
```

## Версия v1.3

## Структура проекта

```text
backend/     - бизнес-логика, работа с БД, расчеты
frontend/    - интерфейс на PySide6/QML
config.ini   - настройки подключения к PostgreSQL
start.bat    - запуск приложения
MachineCostPro.spec - файл сборки PyInstaller
```

## Назначение проекта

MachineCost Pro создавался как рабочий инструмент для производственного учета:
- видеть реальную себестоимость;
- учитывать материалы и труд;
- понимать прибыль по проданным станкам;
- не терять косвенные расходы;
- быстро собирать данные для финансового контроля.

## Статус

Проект активно дорабатывается под реальные производственные процессы.
