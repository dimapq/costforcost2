# MachineCost Pro

Desktop-приложение для учёта производства станков, материалов, сотрудников, финансов и себестоимости.

## Что умеет

- учёт станков: в производстве, на складе, проданные;
- расчёт себестоимости, прибыли и косвенных расходов;
- склад материалов с FIFO, остатками, категориями и ручным редактированием;
- раскрой плит и выпуск деталей из листовых материалов;
- учёт сотрудников, часов, зарплаты, премий и взаиморасчётов;
- финансы: налоги, прочие расходы, экспорт в Excel и SQL-дамп;
- онлайн-подключение к PostgreSQL через `config.ini`;
- встроенные автотесты основных расчётов.

## Технологии

- Python
- PySide6 / QML
- PostgreSQL
- pandas
- openpyxl
- psycopg2

## Запуск из исходников

```bash
python -m venv .venv
.venv\Scripts\python.exe -m pip install -r requirements.txt
start.bat
```

Приложение использует параметры подключения из `config.ini`.

## Сборка

```bash
build_release.bat
```

Результат сборки:

```text
dist/MachineCostPro
dist/MachineCostPro.zip
```
